 ## Project Overview
The Netflix Tableau Analytics Project aims to analyze user viewing patterns, content performance, and engagement metrics using Tableau. The dashboard helps uncover insights into viewer preferences, popular genres, and performance of movies and series across different regions. Interactive visualizations in Tableau support data-driven decision-making for content recommendation, production planning, and business strategy.
 ## Dataset Description
• User ID: Unique identifier for each viewer.
• Show ID: Unique identifier for each movie/series.
• Show Type: Indicates whether content is a Movie or TV Show.
• Title: Name of the show or movie.
• Country: Region where the content is available or viewed.
• Duration: Total run time or number of seasons/episodes.
• Release Year: Year the show/movie was released.
• Ratings: Viewer ratings for each show.
• Total Views: Number of times the content was streamed.
 ## Business Objectives
• Analyze the performance of movies and TV shows based on views and ratings.
• Identify the most watched genres and top-performing countries.
• Understand user preferences by analyzing viewing patterns.
• Track trends based on release year and content type.
• Create an interactive Tableau dashboard for fast and clear insights.
 ## Tools & Techniques Used
Tool:
Tableau — Used for data analysis, visualization, and creation of interactive dashboards.
Data Cleaning:
Ensured consistency, removed missing or duplicate entries, and standardized fields such as duration and ratings
Key Insights
 ## Content Performance
• TV Shows have higher engagement compared to Movies.
• Genres like Drama, Action, and Thriller dominate total views.
• Shows with more seasons generally have higher viewer retention.
 ## Viewer Trends
• Countries like India, USA, and UK contribute the highest viewership.
• Newer content (released after 2018) shows stronger engagement.
• Higher-rated shows correlate with higher total viewing hours.
